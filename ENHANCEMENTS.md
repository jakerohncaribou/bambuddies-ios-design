# Future Enhancements List

Features deferred from initial iOS/Web launch. Tracked here as they come up during mockup review.

## Messaging

### Direct Messages (DMs)
1-to-1 messaging between users outside of game/group context. Not in current web product; removed from Inbox mockup on 2026-04-17.

### New Message Composer (Inbox)
"New" button on Inbox page opens a composer that lets the user initiate a new message to either:
- **A Person** (DM — see above, depends on DM support)
- **A Game** (post into game thread, assuming user is a participant/organizer)
- **A Group** (post into group thread, assuming user is a member)

UI pattern: tap "New" → modal sheet with 3 destination types → select recipient/target → compose message.

Mockup placeholder exists on Page 21 (Inbox) — "New" button visible but not functional.

## Groups

### Discoverable Groups
Toggle on Group Creation that lets a group appear in the public Group Discovery list so strangers can apply to join. Removed from Create Group flow on 2026-04-17. **Note:** Jake has flagged this as *possibly never building* — groups may stay invite-only/private by design. Flagged as future consideration rather than committed roadmap.
